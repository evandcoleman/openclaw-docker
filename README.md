# openclaw-docker

Custom Docker image based on the official [OpenClaw](https://github.com/openclaw/openclaw) image with additional tools.

## What's included

- Everything from `ghcr.io/openclaw/openclaw:latest`
- [GitHub CLI (`gh`)](https://cli.github.com/)

## Usage

```
docker pull ghcr.io/evandcoleman/openclaw-docker:latest
```
